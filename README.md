# 🐾 PetServices - Plataforma de Servicios para Mascotas

## 📌 Descripción

**PetServices** es una aplicación web tipo catálogo que permite a los usuarios explorar, visualizar e interactuar con diferentes servicios orientados al cuidado de mascotas, como veterinaria, grooming, entrenamiento, hospedaje y actividades recreativas.

La plataforma está diseñada con un enfoque en la experiencia del usuario (UX/UI), ofreciendo una interfaz moderna, intuitiva y responsiva.

---

## 🚀 Funcionalidades

* 🔍 Visualización de servicios en tarjetas (cards)
* 📄 Vista detallada de cada servicio
* ❤️ Sistema de favoritos (usando localStorage)
* 📬 Formulario de contacto con validaciones
* ➕ Creación de servicios (Mini CRUD)
* ❌ Eliminación de servicios
* 🌐 Navegación tipo SPA (Single Page Application)
* 📱 Diseño responsivo (adaptable a móvil, tablet y desktop)

## ▶️ Ejecución del Proyecto

### 🔹 Opción 1: Abrir directamente

1. Descargar el proyecto
2. Abrir el archivo:

   ```
   petservices.html
   ```

---

### 🔹 Opción 2: Servidor local (recomendado)

#### Con Python:

```bash
python3 -m http.server 5500
```

Abrir en navegador:

```
http://localhost:5500/petservices.html
```

---

#### Con Node.js:

```bash
npm install -g http-server
http-server
```

Abrir en:

```
http://localhost:8080
```
